# Core package brief

Own pure, environment-neutral behavior:

- exact brand lookup;
- alias/domain resolution;
- ambiguous match handling;
- fuzzy search and scoring;
- category/tag filtering;
- color derivation helpers;
- theme-aware asset selection;
- logical and immutable asset URL construction.

No filesystem, Vue, Cloudflare, or HTTP concerns belong here. Node, browser, and Worker results must match.
